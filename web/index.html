Справка по командам PLC

Описание и примеры
Команда Назначение Пример
TIME Чтение времени контроллера. TIME
TIME ЧЧ:ММ:СС DAY=1..7 DATE=ДД.ММ.ГГ Запись даты и времени в контроллер. TIME 16:05:49 DAY=3 DATE=25.05.26
GEONEZ Чтение широты, долготы и часового пояса. GEONEZ
GEONEZ=LAT,LON,ZONE Запись географических координат. GEONEZ=055,037,03
KEYS Чтение состояния KEYS A/B/C. KEYS
KEYS=ABC Запись состояния KEYS (битовая маска из 0/1). KEYS=101
MASK Чтение маски каналов MASK A/B/C. MASK
MASK=ABC Запись маски каналов (битовая маска из 0/1). MASK=110
COM AAA,CCC Универсальная команда на адрес AAA с кодом CCC. COM 001,010
COMA AAA,CCC Команды сценариев/спектра (legacy протокол). COMA001,011
COMA AAA,010..030 Воспроизведение сценария 1..20. COMA001,015
COMA AAA,031..050 Запись текущего состояния в сценарий 1..20. COMA001,031
COMA AAA,220 Установка стартового сценария при подаче питания. COMA001,220
COMA AAA,140..219 Управление спектральными каналами (R/B/FR/W). COMA001,160
SHED Чтение всего расписания контроллера. SHED
SHED=... Запись одной строки расписания. SHED=09 1 1111111 2-00:10 250 007 0000

Нотация параметров

  • AAA - адрес устройства 000..255.
  • AI - индивидуальный адрес 1..220.
  • AG - групповой адрес 0..29, где AG=0 это широковещательная группа.
  • Важно для записи сцен: используйте AI или AG=1..29. AG=0 (broadcast) может дать непредсказуемую запись спектра.
  • CCC - код команды 000..255.
  • ABC - трехбитная маска каналов A/B/C.

Расшифровка COMA по числам

  • Воспроизведение сцены: CCC = 10 + N, где N = 1..20. Пример: 015 = сцена 5.
  • Запись сцены: CCC = 30 + N, где N = 1..20. Пример: 035 = записать в сцену 5.
  • Каналы спектра (BASE): 140 = красный, 160 = синий, 180 = дальний красный, 200 = белый.
  • Код канала: CCC = BASE + S, где S = шаг мощности 0..19.
  • Преобразование процента в шаг S: 0% -> 0; 1..10% -> 1; 11..100% -> floor(%/5)-1.
  • Пример 161: 160 (синий BASE) + 1 (шаг) -> синий 10%.
  • Пример 143: 140 (красный BASE) + 3 (шаг) -> красный 20%.
  • Пример 209: 200 (белый BASE) + 9 (шаг) -> белый 50%.
  • Пример 180: 180 (дальний красный BASE) + 0 (шаг) -> дальний красный 0%.

Чтение времени:
TIME

Запись времени:
TIME 08:30:00 DAY=1 DATE=20.04.26

Сохранить полный спектр (R/B/W/FR) в сценарий 5:
COMA001,143
COMA001,165
COMA001,209
COMA001,180
COMA001,035

Запустить сценарий 5:
COMA001,015

Записать строку расписания:
SHED=00 1 1111111 0+08:00 001 011 0000
