# PLC Nova AI Agent

AI-агент для управления PLC-контроллерами освещения через естественный язык с поддержкой голосового ввода.

## Возможности

- 🧠 **LM Studio API** - управление моделями через локальный сервер LM Studio
- 🎤 **Голосовой ввод** - распознавание речи через SpeechRecognition (Google API)
- 💬 **Чат-интерфейс** - полноценный диалог с историей сообщений
- ⚡ **Быстрые команды** - управление светильниками в один клик
- 🔧 **Tools & Skills** - расширяемая архитектура

## Установка

### 1. Зависимости

Установите [LM Studio](https://lmstudio.ai/) и запустите локальный сервер.

**Преимущества:**
- GPU ускорение из коробки
- Простое переключение между моделями
- Поддержка MCP и плагинов
- Нет необходимости в llama-cpp-python

### 2. Модели

#### LLM через LM Studio
1. Установите LM Studio
2. Откройте LM Studio и загрузите модель через GUI
3. Запустите локальный сервер (по умолчанию `http://127.0.0.1:1234`)

Поддерживаемые форматы: GGUF, mlx

#### SpeechRecognition (голосовой ввод)
Голосовой ввод работает через Google Speech API (требует интернет):
```bash
pip install SpeechRecognition pyaudio
```

Нет необходимости скачивать модели - используется облачный API Google.

## Использование

### Через веб-интерфейс

1. Откройте вкладку **"ИИ"** в приложении
2. Убедитесь что LM Studio запущен и модель загружена
3. Включите микрофон для голосового ввода
4. Введите команду или скажите её

### Примеры команд

```
- "Включи свет на 50%"
- "Запусти сцену 5"
- "Красный канал 80%, синий 30%"
- "Как настроить расписание?"
- "Прочитай время контроллера"
```

### Программный доступ

```python
from agent.main import get_agent

agent = get_agent()

# Проверка доступности LM Studio
agent.check_lm_studio()

# Загрузка модели через LM Studio
agent.load_model("gemma-3-270m-it-qat")

# Выполнение команды
result = await agent.execute("Включи свет")

# Голосовое распознавание
with open("audio.wav", "rb") as f:
    audio_data = f.read()
text = agent.recognize_speech(audio_data)

# Генерация ответа через LLM
response = agent.generate_response("Как дела?")
```

## Конфигурация

### Параметры модели

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| context_size | 4096 | Размер контекстного окна |
| max_tokens | 512 | Максимум токенов в ответе |
| temperature | 0.7 | Креативность (0.1-2.0) |
| top_p | 0.9 | Сэмплирование |
| repeat_penalty | 1.1 | Штраф за повторы |

### Параметры SpeechRecognition

| Параметр | Значение | Описание |
|----------|----------|----------|
| sample_rate | 16000 | Частота дискретизации |
| language | ru-RU | Язык распознавания |

## Архитектура

### Tools
Инструменты для взаимодействия с устройством:
- `plc_command` - отправка произвольных команд
- `scene_control` - управление сценами и каналами
- `voice_input` - распознавание речи

### Skills
Скиллы для обработки естественного языка:
- `plc_control` - управление светильниками
- `ai_chat` - диалог с LLM

Скиллы загружаются из Markdown-файлов с frontmatter.

## API Endpoints

| Endpoint | Описание |
|----------|----------|
| `api_agent_status` | Статус агента |
| `api_agent_execute` | Выполнение команды |
| `api_agent_load_model` | Загрузка модели |
| `api_agent_unload_model` | Выгрузка модели |
| `api_agent_clear_history` | Очистка истории |
| `api_voice_recognize` | Распознавание речи |
