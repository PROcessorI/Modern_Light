# PLC Control Skill

Скилл для управления светильниками PLC через естественный язык.

## Возможности

- Включение/выключение света
- Установка яркости (0-10), где 1 - это 10% яркости
- Активация сценариев (1-20)
- Запись сценариев
- Управление спектральными каналами
- Отправка произвольных команд COM/COMA

## Примеры команд

### Яркость
- "Установи яркость 5"
- "Яркость 50%"
- "Выключи свет" (яркость 0)
- "Включи свет на полную" (яркость 10)

### Сценарии
- "Запусти сцену 5"
- "Активируй сценарий 3"
- "Запиши текущий спектр в сцену 7"
- "Включи стартовую сцену"

### Спектральные каналы
- "Красный канал 50%"
- "Синий 25%"
- "Белый на полную"
- "Дальний красный 0%"
- "Установи все каналы: красный 50, синий 30, белый 80"

### COM/COMA команды
- "Отправь COM 001,010"
- "COMA 001,220"
- "Прочитай время"
- "Установи время 12:00:00 DAY=1 DATE=25.12.24"

## Обработка команд

Скилл распознаёт следующие паттерны:

1. **Яркость**: "яркость N", "свет N", "включи N"
2. **Сцены**: "сцена N", "сценарий N", "запустить N"
3. **Каналы**: "<цвет> N%", "<цвет> канал N"
4. **COM**: "COM AAA,CCC"
5. **COMA**: "COMA AAA,CCC"

## Адресация

По умолчанию используется:
- AI=1 (индивидуальный адрес)
- AG=0 (групповой адрес, широковещание)

Можно указать явно:
- "Для светильника 5 яркость 7"
- "Группе 3 сцену 2"
