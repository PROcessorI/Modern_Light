# AI Chat Skill

Скилл для общения с локальной GGUF-моделью через естественный язык.

## Возможности

- Диалог с AI-моделью в контексте управления PLC
- Автоматическое извлечение команд из естественного языка
- Объяснение настроек и параметров
- Помощь в конфигурировании сценариев

## Примеры команд

### Диалог
- "Как настроить расписание?"
- "Что делает команда COMA 220?"
- "Помоги настроить сцену для рассады"
- "Объясни разницу между AI и AG адресами"

### Запросы с действием
- "Спроси у модели как лучше настроить свет для томатов"
- "Что если я включу все каналы на 100%?"

## Контекст

Модель имеет контекст о:
- Протоколе PLC (COM, COMA, TIME, GEONEZ, и т.д.)
- Спектральных каналах (красный, синий, дальний красный, белый)
- Сценариях (1-20)
- Расписании

## Обработка

Скилл отправляет запрос пользователя в LLM вместе с системным промптом
о возможностях системы PLC. Модель генерирует ответ на естественном языке.

Если в ответе модели содержится команда PLC, она может быть автоматически
выполнена через tools.
